# Rock Paper Scissors Game

A simple and fun **Rock Paper Scissors** web app built with **HTML, CSS, and JavaScript**.  
Play against the computer, track your score using local storage, or let the game play automatically in Auto Play mode!


## Features

- Play Rock, Paper, or Scissors against the computer  
- Randomized computer moves  
- Score tracking (saved in local storage)  
- Auto Play mode (computer plays for you)  
- Keyboard shortcuts:
  - **R** → Rock  
  - **P** → Paper  
  - **S** → Scissors  
  - **Backspace** → Reset the score
